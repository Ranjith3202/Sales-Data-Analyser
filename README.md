📊 Sales Data Analytics Dashboard with Machine Learning

 Project link - https://huggingface.co/spaces/Ranjith511/Data-Analyser
🔹 Overview

This project is an end-to-end data analytics dashboard built using Python and Streamlit. It allows users to upload datasets, clean data, visualize insights, store data in MySQL, and predict future sales using machine learning.

🔹 Features

📁 Upload CSV/Excel datasets
🧹 Data cleaning (remove duplicates, handle missing values)
📊 Interactive visualizations (Bar, Line, Pie charts)
💾 Store cleaned data in MySQL database
📥 Download cleaned dataset
🤖 Machine Learning prediction (Linear Regression)
📅 Forecast sales for future days

🔹 Technologies Used

Python
Streamlit
Pandas & NumPy
Matplotlib
MySQL
Scikit-learn

🔹 Machine Learning

Algorithm: Linear Regression
Converts date into numeric format
Trains on historical sales data
Predicts future sales dynamically

🔹 Use Case

This project helps businesses:

Clean and process raw data
Analyze trends
Forecast future sales
Make data-driven decisions
